## Branding Guidelines for Caltrain Compliance

### ✅ ALLOWED:
- Descriptive use: "Real-time tracker for Caltrain"
- Generic references: "Caltrain schedule app"
- Attribution: "Data provided by Caltrain"

### ❌ NOT ALLOWED (per Developer License):
- Using "Caltrain" in your brand name without approval
- Using Caltrain logos or confusingly similar variants
- Implying official endorsement
- Using Caltrain trademarks in domain name

### Recommended Approach:
- Brand: **TrackTrain**
- Domain: **tracktrain.app**
- Tagline: "Real-Time Tracker for Caltrain"
- Clear disclaimer: "Independent app, not affiliated with Caltrain"
